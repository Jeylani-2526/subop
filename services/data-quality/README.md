# Module 7 — Data Quality Engine

**Status:** ⏳ Scheduled for M10 (29 Dec 2026 – 2 Feb 2027)
**Owner:** Abdullah

## Description

Automatically validates data against configurable rules and generates a quality score
for every dataset. Supported checks: null, duplicate, format, range, and anomaly.

## KPI Target

Quality rules run automatically on every pipeline execution.
A quality score is generated per dataset and anomalies are flagged with alerts.
