# Module 10 — Security & Compliance

**Status:** ⏳ Scheduled for M11 (3 Feb – 23 Mar 2027)
**Owner:** Omer (implementation) + Abdullah (KVKK/GDPR documentation)

## Description

Cross-cutting security module enforced at the FastAPI API layer across all platform modules.

## Features

| Feature | Description |
|---------|-------------|
| RBAC | Role-based access control for 4 roles: Data Engineer, BI Analyst, Platform Admin, Viewer |
| Column Masking | Server-side masking — enforced at API layer, never client-side |
| Anonymisation | Data anonymisation for sensitive fields |
| Audit Logs | Full audit log of every user and system action |
| KVKK / GDPR | Compliance documentation and gap analysis |
